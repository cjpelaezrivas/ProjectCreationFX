In this page you can find all the references that I considered appropiate to take into account or links that I have used to create this project.

Project references

  • Minlog [Modified]
    Tiny Java logging library modified and, in my opinion, improved.
  • TinySound
    Simple sound system that wraps the standard Java sound libraries.
  • JOrbis
    Pure Java Ogg Vorbis decoder.
  • VorbisSPI
    Java Service Provider Interface that adds OGG Vorbis audio format support to Java platform.
  • Tritonus Share
    Implementation of the Java Sound API.
  • GitHub Java API (org.eclipse.egit.github.core)
    Java library for communicating with the GitHub API. Part of the GitHub Mylyn Connector.
  • google-gson
    Java library that can be used to convert Java Objects into their JSON representation. It can also be used to convert a JSON string to an equivalent Java object.

Other references

  • JavaDevOne’s Java 2D game
    Basic 2D game. The idea of starting Project Creation FX comes from this project.
  • Almas Baimagambetov’s FXGL
    Game Development Framework. Some concept ideas taken from his sites and projects.
  • The nature of code
    A really interesting book that codes some nature concepts and behaviors. The idea of putting in practice some concepts found this books is one of the main motivations of Project Creation FX.
